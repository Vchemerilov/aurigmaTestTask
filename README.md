Тестовое задание от aurigma - разработать файловый веб-менеджер.

Серверная часть разработана с применением технологии ASP.NET MVC.
Клиенткая часть - jQuery. Для верстки использовался Bootstrap (модальное окно).

Путь к папке указывается в web.config - значение pathToFolder. По умолчанию d:\\dir\\.
Максимальный размер файла, закачиваемого на сервер - 100000 КБ. Указан параметром maxRequestLength в том же файле. 

JS файл с аяксовыми запросами находится по пути /Scripts/js/. CSS-файл - /css/.
